# TyreTrack — Premium Car Service Website

A cinematic, glass-dark themed website for **TyreTrack** garage with red/black luxury styling inspired by Apple, Tesla, and Logic Pro aesthetics.

## Features

- Cinematic hero with spinning tyre animation and scroll-driven navbar
- Services: Wheel Alignment, Tyre Services, Water Wash, Interior Cleaning
- Booking form with local storage
- Phone login (demo OTP: `123456`)
- Live service tracking (demo ID: `TT-2026-4821`)
- Before/after gallery with drag compare
- Contact page with Google Maps embed

## Run locally

```bash
cd tyretrack
npm install
npm run dev
```

Open the URL shown in the terminal (usually `http://localhost:5173`).

## Replace demo data

Edit `src/data/services.ts`, `gallery.ts`, and `tracking.ts` with your real phone, address, map embed URL, images, and pricing.
